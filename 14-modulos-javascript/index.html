Módulos Javascript

  • Exportação nomeada (padrão)

    1. Ao usar a tag <script> no HTML, especifique o type como "module"
    <script type="module" src="index.js"></script>
    1. Crie um arquivo js e construa uma função usando export
    export function somar(num1, num2) {
            return num1 + num2;
        }
    1. No arquivo onde queira usar a função exportada, use o import de acordo com código abaixo:
    import { somar } from './nome-do-arquivo.js'
    1. Pronto agora é só chama-la!
    console.log(somar(1, 3))

  • Exportação default (Não aconselhável)

    É possível trocar o nome na hora da importação, se utilizarmos o default no export.

    function teste(num1, num2) {
            return num1 + num2;
        }

        export default teste

    Não use as chaves {} ao importar a função com nomes diferentes.

    import somar from './nome-do-arquivo.js'
